# Diwas Upadhyay — Portfolio

Personal developer portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Tech Stack

- **React 19** + **Vite 7** — fast dev & build
- **Tailwind CSS v4** — utility-first styling
- **Framer Motion** — animations & transitions
- **Canvas API** — interactive cat & yarn game

## Features

- Animated hero with scroll-aware progress bar cat
- Glassmorphic project cards with hover effects
- Interactive cat & yarn ball game in the footer
- Konami code easter egg (↑↑↓↓←→←→)
- Particle effects, aurora blobs, and mouse spotlight
- Fully responsive

## Getting Started

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Deployed on [Netlify](https://netlify.com). The `netlify.toml` config is included.
